# DMedak Portfolio

Personal software development portfolio built to showcase projects, skills and ongoing work in web and game development.

The goal of this project is to create a modern and visually polished portfolio experience that represents my approach to software engineering, UI design and interactive applications.

This portfolio is currently in active development and will continue evolving with new features, projects, animations and overall UI improvements.

## Running the Project

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```txt
http://localhost:3000
```

## Status

This project is currently under development.
Additional features, animations, sections and project showcases will be implemented throughout the development process.
This portfolio is being developed as a long-term hands-on project focused on real problem solving, iterative development and building modern frontend experiences through practical implementation and continuous improvement.
